## 二、核心功能

### 1. 语料加载
- 从 JSON 文件加载原始语料
- 自动转换为带元数据的数据结构
- 支持增量更新

### 2. 智能检索
- 按分类 + 子分类检索
- **权重随机选择**：效果好(高评分)的优先选中
- 支持条件过滤（用户群体、情绪强度、对话阶段）

### 3. 自动扩展
当低性能语料过多时自动扩展：

| 策略 | 示例 |
|------|------|
| 同义词替换 | "我在听" → "我正在听" |
| 语气调整 | "你慢慢说" → "你慢慢说，真的" |
| 共情前缀 | + "我理解，" + 原句 |
| 疑问转换 | "我在听。" → "我在听吗？" |

### 4. 评价机制

#### 显式评价（用户直接反馈）
```
用户反馈按钮: 👍 有帮助 | 😐 一般 | 👎 没帮助
↓
评分更新: score = score * 0.7 + feedback * 0.3
```

#### 隐式评价（行为分析）
```
用户行为分析:
- 继续发送消息 → 正向 (+0.2)
- 立即结束对话 → 负向 (-0.2)
- 停留时间过长 → 中性
```

### 5. 定期任务
- **频率**：每天/每周
- **任务**：
  1. 识别低性能语料 (score ≤ 0.3)
  2. 自动扩展变体
  3. 输出优化建议
  4. 统计更新

## 三、数据结构

```typescript
interface CorpusItem {
  content: string;           // 语料内容
  category: string;           // 主分类
  subcategory: string;        // 子分类
  usageCount: number;        // 使用次数
  effectivenessScore: number; // 效果评分 0-1
  createdAt: number;         // 创建时间
  lastUsedAt?: number;       // 最后使用时间
}

interface CorpusStats {
  totalItems: number;        // 总条数
  averageScore: number;      // 平均评分
  highPerformanceItems: number; // 高性能 (>0.7)
  lowPerformanceItems: number; // 低性能 (<0.3)
}
```

## 六、使用示例

```typescript
import { corpusManager, initializeCorpus, corpusEvaluator } from './CorpusManager';

// 1. 初始化（应用启动时）
initializeCorpus(corpusData);

// 2. 获取语料
const item = corpusManager.getItem('01_通用安抚承接语', '情绪确认类');
if (item) {
  corpusManager.recordUsage(item);
  return item.content;
}

// 3. 用户评价反馈
corpusEvaluator.processFeedback(item, 'like'); // 显式评价
corpusEvaluator.implicitEvaluate(item, true);  // 隐式评价

// 4. 查看统计
const stats = corpusManager.getStats();
console.log(`总语料: ${stats.totalItems}, 平均分: ${stats.averageScore}`);
```
